I build **data-intensive web apps** for engineering and scientific domains — interactive dashboards, canvas editors, and calculation interfaces. Currently working as Lead Developer at a railway research institute (Moscow), finishing an MSc in Information Systems (NUST MISIS, 2026).

**What I actually do at work:**
- Plotly.js / Konva.js / Chart.js visualization layers — graphs, canvas editors, comparison modes
- Full-stack features: React/TypeScript front + Node.js REST API + PostgreSQL
- Server setup & containerised deployment (Nginx, Podman, Docker)
- UI/UX redesign and prototyping in Figma Make

**Open source:** I published [guide-kit](https://krokodila888.github.io/guide-kit-docs/) — an onboarding component library for complex professional software (React / Vue / vanilla JS adapters). It was my MSc thesis and is now used in production.
